PURPOSE

function [im] = nonmax(im,theta)

SYNOPSIS

function [im] = nonmax(im,theta)

DESCRIPTION 

 function [im] = nonmax(im,theta)

 Perform non-max suppression on im orthogonal to theta.  Theta can be
 a matrix providing a different theta for each pixel or a scalar
 proving the same theta for every pixel.
